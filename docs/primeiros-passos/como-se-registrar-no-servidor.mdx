# Como se registrar no servidor

Aprenda como se registrar em nosso servidor

## Como se registrar no Minecraft

Depois de entrar em nosso servidor, você precisará se registrar para poder jogar. Isso é feito através do comando `/register`. Quando você digita este comando, você será solicitado a digitar sua senha duas vezes, para que possamos confirmar que você digitou corretamente. Por exemplo, se você quiser definir a senha "senha123", você deverá digitar o seguinte:

```cmd title="Comando para ser usado no chat do Minecraft"
/register senha123 senha123
```

Observe que é importante que você digite a senha corretamente duas vezes, caso contrário, você não será capaz de se registrar. Além disso, tenha em mente que você precisa usar uma senha segura e única, que não seja fácil de ser adivinhada.

Se você já possui uma conta original no Minecraft Java ou no Bedrock, você não precisará se registrar. Basta fazer login com sua conta original e você será logado automaticamente em nosso servidor. Se tiver alguma dúvida sobre como se registrar ou fazer login, por favor, não hesite em entrar em contato conosco através de nosso discord. Estamos sempre aqui para ajudar.
