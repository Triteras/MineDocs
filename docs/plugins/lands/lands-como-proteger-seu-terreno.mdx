# Lands

Bem-vindo à sub-seção do plugin de terreno no nosso servidor Minecraft Triteras. Aqui você aprenderá como usar o criar seu terreno.

## Como proteger seu terreno

Clamar uma terra no Lands é uma etapa importante para garantir que suas construções e recursos estejam protegidos. Aqui está um pequeno passo a passo para usar o comando `/lands claim`:

1. `/lands claim`: com este comando, você pode reivindicar uma área específica usando uma enxada de ouro. Para isso, selecione dois pontos diagonais em um formato quadrado ou retangular para selecionar a área que deseja reivindicar. Após a seleção, digite o comando e pronto, você reivindicou sua área.

2. `/lands unclaim`: com este comando, você pode desocupar uma área específica que já foi reivindicada por você. Basta selecionar a área que deseja liberar usando a enxada de ouro e digitar o comando. A área será desocupada e poderá ser reivindicada por outro jogador.

3. `/lands unclaimall`: com este comando, você pode liberar todas as áreas que você reivindicou, liberando assim toda a sua terra. Este comando é útil quando você deseja desocupar completamente todas as áreas que possui.

4. `/lands claim auto`: com este comando, você pode reivindicar automaticamente a área da chunk em que você está no momento, sem a necessidade de selecionar manualmente a área com a enxada de ouro. Para isso, basta digitar o comando e a chunk atual será reivindicada para você. Basta você andar por cima do terreno clamado.

5. `/lands unclaim auto`: com este comando, você pode desocupar automaticamente a área da chunk em que você está no momento, liberando assim a área para que outros jogadores possam reivindicá-la. Para isso, basta digitar o comando e a chunk atual será desocupada. Basta você andar por cima do terreno clamado.

Lembre-se de que, para utilizar esses comandos, é necessário ter permissão e MineCoins, que são a moeda utilizada pelo plugin Lands. Com esses comandos, você pode reivindicar terras, liberá-las e gerenciá-las de forma fácil e rápida
Para mais informações use o comando `/lands menu` para abrir o menu interativo do Lands.
